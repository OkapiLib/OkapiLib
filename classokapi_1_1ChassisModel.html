okapi::ChassisModel class
#include <include/okapi/api/chassis/model/chassisModel.hpp>

A version of the ReadOnlyChassisModel that also supports write methods, such as setting motor speed. Because this class can write to motors, there can only be one owner and as such copying is disabled.

void okapi::ChassisModel::setMaxVelocity(double imaxVelocity) pure virtual

Parameters
imaxVelocity The new maximum velocity.

Sets a new maximum velocity in RPM. The usable maximum depends on the maximum velocity of the currently installed gearset. If the configured maximum velocity is greater than the attainable maximum velocity from the currently installed gearset, the ChassisModel will still scale to that velocity.
