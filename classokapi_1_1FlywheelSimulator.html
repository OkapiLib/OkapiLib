Function documentation

okapi::FlywheelSimulator::FlywheelSimulator(double imass = 0.01, double ilinkLen = 1, double imuStatic = 0.1, double imuDynamic = 0.9, double itimestep = 0.01) explicit

A simulator for an inverted pendulum. The center of mass of the system changes as the link rotates (by default, you can set a new torque function with setExternalTorqueFunction()).

void okapi::FlywheelSimulator::setExternalTorqueFunction(std::function<double(double angle, double mass, double linkLength)> itorqueFunc)

Parameters
itorqueFunc the torque function. The return value is the torque due to external forces

Sets the torque function used to calculate the torque due to external forces. This torque gets summed with the input torque.

For example, the default torque function has the torque due to gravity vary as the link swings: [](double angle, double mass, double linkLength) { return (linkLength * std::cos(angle)) * (mass * -1 * gravity); }
