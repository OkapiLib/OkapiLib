Function documentation

okapi::ChassisController::ChassisController() explicit defaulted

A ChassisController adds a closed-loop layer on top of a ChassisModel. moveDistance and turnAngle both use closed-loop control to move the robot. There are passthrough functions for everything defined in ChassisModel.
